Animative- Image Hover Effects is a HTML5 & CSS3 responsive bootstrap snippet. User can easily use grid design in there website, plugin etc. In this snippet, there are Thirty(30) types of hover effects with different layouts. It is fully responsive, featured rich and beautifully designed. It is a multipurpose image hover effects (i.e you can used it in your websites or anywhere in your projects). Easily customizable, 24/7 support time.

General Information

Software requirements
  • Sublime Text 3 used to edit the .HTML, .CSS and .SCSS files.
  • Grunt used as precompiler for SCSS to CSS convert.

The following scheme displays the theme files structure.

  • Documentation - contains documentation on snippet cusotmisation and support
  • snippet - contains the main files of the snippet. which will be uploaded on server.
    • CSS - contains all Stylesheets used for UI.
    • Font Awesome - contains Font awesome files.
    • Images - contains all the images.
    • SASS - contains all SCSS and variable files.
    • Index.html - contains all UI code.
  • Screenshot- Contains screenshot of snippet.
  • readme.txt- contains the information about snippet.
